# ⚽ FootballMonk

A full-stack JerseyWala jersey e-commerce application built for a freelance client.

🔗 **Live Demo:** [JerseyWala.vercel.app](https://JerseyWala.vercel.app)

---

## ✨ Features

- 🔐 Authentication (Supabase Auth + Google OAuth)
- 🛍️ Product listing with search & category filter
- 🛒 Cart with size selection & quantity management
- 💳 Razorpay payment gateway integration
- 📦 Order tracking with real-time status
- 🔔 User notifications
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

| Layer      | Technology            |
| ---------- | --------------------- | ------- |
| Frontend   | React 19 + TypeScript | Tanstck |
| Styling    | Tailwind CSS v4       |
| Routing    | React Router v7       |
| Backend/DB | Supabase (PostgreSQL) |
| Auth       | Supabase Auth         |
| Payment    | Razorpay              |
| Deploy     | Vercel                |

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/abdalqamar/JerseyWala.git
cd JerseyWala

# 2. Install dependencies
npm install

# 3. Setup environment variables
cp .env.example .env
# Fill in your Supabase and Razorpay keys

# 4. Start development server
npm run dev
```
